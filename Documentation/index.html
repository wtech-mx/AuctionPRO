About

"AuctionPRO" is a complete solution for an online Auction Platform. Anyone can register and enlist their product for auction for basic or featured fee. Admin will get profit from the basic and featured fee as well as from the withdraw percentage. Everything is dynamic and can setup from admin panel. This CMS Includes almost everything you need to make an online auction platform and fully responsive for any device. Admin can manage complete website without single line of coding knowledge. It has strong SQL injection protection system which will keep away this system from hackers.

Installation

Download the file from codecanyon then extract in your computer and you will get three folders.

Step 1 : Extract files.zip file from Files folder to server.

Step 2 : Create database using database.sql from Database folder.

Step 3 : Edit details of .env file in project folder in server.

Few Server has problem with .env file. If you face any problem to find the .env file then you may need to enable to show hidden files in server or check how to add .env file in your server. We are also available to help.

After installation you can login in the admin panel by this link:

Dashboard

The dashboard is very simple, informative and dynamic. Admin can change his account details from top right corner. So admin doesn't need to do this manually from database.

Auctions

All the Auctions will be here. Admin can view details, add/edit and delete too.

Withdraws

All the Withdraw requests will be here. Admin can view details and process it.

Category

All the Category will be here. Admin can view edit, delete or add from here.

Manage Users

All the users will be here. Admin can view details or ban them

Staffs

Admin can add more Staffs here.

Language Settings

You can change your website language from admin panel. Just write the website text in your language. You can input any UTF-8 Language. If you face any problem then contact us.

Theme Color

You can change complete theme color within few seconds. Just input your color or select color from color picker.

Slider Settings

Admin Can add Slider image, title and slider text from here. Slider text can be decorated as Righ, left of middle position.

Service Section

Admin can add or edit from this section. Title and Title bottom text can be added from here. Everything is dynamic in this system.

Portfolio Section

Admin can add or edit from this section. Title and Title bottom text can be added from here. Everything is dynamic in this system. Admin can select multiple images at a time and upload single image too.

Blog Section

Admin can add or edit blog from this section. All the featured images in blog must be same size. It will look good and will show perfectly.

Testimonial Section

Admin can add or edit from this section. Title and Title bottom text can be added from here. Everything is dynamic in this system. Admin must add atleast 3 testimonials to run this section correctly.

Page Settings

You can Enable or Disable About Us, FAQ and Contact us Page. Just need a single click. You can add multiple email for the Contact Us page email address. Just separate emails using comma and don't give any space.

Social Settings

This script allows you to add social links in your website very easily. You just need to put the link. If you want to disable any social media then you can do it by single click and it will not appear in the Website. You can Enable it again by single click.

Advertisements

This script offers two type of Advertisement support. You can add advertisement by code or you can add advertisement by banner and advertisement URL. To add advertisement first click on Add New Advertisement button then select Advertisement type then complete the simple steps.

Administrator can Edit, Deactivate or Remove advertisements by single click and Deactivated advertisements can be Activated by single click too.

SEO Tools

Google Analytics

This script offers you easily Google Analytics integration system, it is a freemium web analytics service offered by Google that tracks and reports website traffic. You just need to put your Google Analytics script in the form and everything will be done.

Meta Keywords

You can add meta keywords in your website by writing them in this form. Every keyword must be separated using comma. It will help you for better SEO.

N.B. Please don't use any space after comma.

General Settings

You can setup logo, icon, title, content, footer area and payment information from here.

Subscribers

You can see all the subscribers of your website in the Subscribers section. You can download the list by single click in .csv format.
